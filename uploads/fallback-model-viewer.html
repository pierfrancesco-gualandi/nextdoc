Impossibile visualizzare il modello

Il formato del modello o la struttura dei file non è supportata direttamente nel visualizzatore integrato.

Opzioni disponibili:

  • Verifica che tutti i file necessari siano stati caricati correttamente
  • Prova a caricare il modello in formato GLB/GLTF (consigliato)
  • Se il modello è in formato HTML/WebGL, verifica la struttura delle cartelle
